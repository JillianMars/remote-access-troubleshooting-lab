# Remote Access Troubleshooting Lab

This repository contains structured troubleshooting simulations focused on diagnosing and resolving remote access failures within a Windows environment.

Each lab intentionally introduces a controlled fault, applies layered diagnostic methodology, isolates root cause, and validates remediation.

## Labs Included

- DNS Misconfiguration
- VPN Hostname Resolution Failure  
- Windows Firewall Outbound Rule Blocking VPN Connectivity  

## Skills Demonstrated

- Windows networking diagnostics  
- DNS resolution analysis  
- Windows Defender Firewall rule configuration  
- VPN (IKEv2) client behavior troubleshooting  
- Outbound traffic policy evaluation  
- Root cause isolation and validation  
- Structured troubleshooting methodology
